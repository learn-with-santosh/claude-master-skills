# SEO Content Master 📈

This skill is designed to produce content that both search engines love and humans enjoy reading. It focuses on semantic SEO, user intent, high-authority structure, and conversion optimization.

## When to use this skill

Use this skill when:
- Writing blog posts, landing pages, or technical documentation.
- Optimizing existing content to improve its Google ranking.
- Developing a content strategy for a new project.
- Creating "Masterclass" guides or pillar pages.

## Content Framework: The "Authority Engine"

### 1. Intent Mapping
Before writing, determine the user's intent:
- **Informational**: "How to...", "What is..."
- **Transactional**: "Best X for Y", "Pricing for Z"
- **Navigational**: "Log in to X"

### 2. Semantic Structure
- **H1**: Single, high-impact title with primary keyword.
- **H2-H3**: Logical hierarchy that covers all sub-topics (the "Topic Cluster" approach).
- **Entities**: Include related terms and concepts that search engines expect (e.g., for "React," mention "Hooks," "JSX," "Virtual DOM").

### 3. Engagement & Retention
- **The Hook**: Grab attention in the first 100 words.
- **Scannability**: Short paragraphs, bullet points, and bold text.
- **Internal Linking**: Strategically link to other pages to keep users on-site.

## Output Format

### 🎯 Strategy & Keywords
Primary keyword, secondary keywords, and target intent.

### 📜 Content Draft
The optimized, high-authority text.

### 🛠 SEO Meta-Data
Optimized Title Tag and Meta Description.

### 🚀 Optimization Score
Explanation of why this content will rank (keyword density, entity coverage, readability).

## Examples

### Example 1: Optimizing a Hook
**Input**: "I'm writing about Next.js 14."
**Output**: 
"Instead of 'Next.js 14 is out', let's use a power hook: 'Next.js 14 is here, and it's changing how we think about Server Actions. But is it worth the migration?' (Intent: Informational + Curiosity)."

### Example 2: Entity Coverage
**Input**: "Topic: AI Coding Assistants"
**Output**:
"To rank for this, we must include these entities: 'LLMs', 'Context Window', 'Claude Code', 'GitHub Copilot', 'Prompt Engineering', and 'Developer Productivity'."

## Best Practices
- **Write for Humans First**: Google rewards engagement. If it's boring, it won't rank.
- **Use Semantic HTML**: Ensure `<h1>` through `<h3>` are used correctly.
- **Optimize Images**: Always remind the user to include ALT text and compressed images.
